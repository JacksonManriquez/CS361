# Students Activity Calendar

A React application for managing student activities with a calendar view and event list.

## Features

- Monthly calendar view with current month display
- Event list
- CRUD placeholders (to be implemented)
- Responsive design

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Technologies

- React
- Vite
- CSS
